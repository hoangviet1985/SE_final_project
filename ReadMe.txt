OCR using Matlab

To Run:

Install Ghostscript for converting pdf files to jpg:
https://ghostscript.com/download/

Install Matlab webcam package for using webcam to take pictures
Follow instructions here:
https://www.mathworks.com/help/supportpkg/usbwebcams/ug/installing-the-webcams-support-package.html

Open gui.m with Matlab
To use analyze a png/jpg, use browse image in the top left and then hit analyze image
To convert a pdf to jpg, use browse in the bottom left
To take a picture with your webcam, turn the Camera on and then take image
